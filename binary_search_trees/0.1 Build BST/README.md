# 0.1 Build a Binary Search Tree

## English version

### Problem
Given a sequence of integers, build a **binary search tree (BST)** by inserting keys in the given order. Then perform a **straight left-order traversal** of the tree.

All keys in a BST are unique. If a key already exists in the tree, it is **ignored** during insertion.

### Input
`input.txt` contains integers — one per line — representing keys to be inserted in the BST.

### Output
`output.txt` should contain the keys in the order obtained by **straight left-order traversal**, one per line.

### Constraints
- Time limit: **1 second**
- Memory limit: **256 MB**

# 0.1 Построить бинарное поисковое дерево

## Русская версия

### Условие
Дана последовательность целых чисел. Постройте **бинарное поисковое дерево (БПД)** по порядку вставки ключей. Выполните **прямой левый обход** дерева.

Все ключи в БПД уникальны. Если ключ уже присутствует в дереве, он **игнорируется** при добавлении.

### Входные данные
Файл `input.txt` содержит целые числа — по одному в строке, в порядке добавления в дерево.

### Выходные данные
Файл `output.txt` должен содержать ключи в порядке прямого левого обхода дерева, по одному числу на строку.

### Ограничения
- Время: **1 секунда**
- Память: **256 МБ**
