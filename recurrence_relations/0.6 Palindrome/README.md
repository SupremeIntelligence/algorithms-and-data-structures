# 0.6 Palindrome

## English version

### Problem
Given a non-empty string \(S\) of at most 7000 lowercase Latin letters, remove the **minimum number of characters** to make it a **palindrome** (a string that reads the same forwards and backwards).

### Input
The input file `input.txt` contains a single line with the string \(S\).

### Output
- First line: the **length of the resulting palindrome**.  
- Second line: the **palindrome itself**. If there are multiple palindromes of maximal length, output any one.

### Constraints
- Time limit: **1 second**  
- Memory limit: **not specified**

# 0.6 Палиндром

## Русская версия

### Условие
Вводится непустая строка \(S\) длиной не более 7000 символов, состоящая из строчных латинских букв.  
Необходимо удалить **минимальное число символов**, чтобы получился **палиндром** (строка, одинаково читающаяся слева направо и справа налево).

### Входные данные
Файл `input.txt` содержит одну строку — исходную строку \(S\).

### Выходные данные
- Первая строка: длина получившегося палиндрома.  
- Вторая строка: сам палиндром. Если палиндромов несколько, вывести любой.

### Ограничения
- Ограничение по времени: **1 секунда**  
- Ограничение по памяти: **не указано**
