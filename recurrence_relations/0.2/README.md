# 0.2 Ones (Part 1)

## English version

### Problem
Given two integers **N** and **K**, determine how many **binary strings of length N** contain **exactly K ones**.

A binary string consists only of characters `0` and `1`.

The result can be very large, therefore it must be printed **modulo 10⁹ + 7**.

### Input
The input contains two integers:
`N K`

Constraints:
`0 ≤ K ≤ N ≤ 1000`

### Output
Print a single integer — the number of binary strings of length **N** containing exactly **K ones**, taken **modulo 10⁹ + 7**.

### Constraints
- Time limit: **1 second**
- Memory limit: **256 MB**

# 0.2 Единицы (часть 1)

## Русская версия

### Условие
Даны два целых числа **N** и **K**.

Необходимо определить, **сколько существует бинарных строк длины N**, содержащих **ровно K единиц**.

Бинарная строка состоит только из символов `0` и `1`.

Так как ответ может быть очень большим, его необходимо вывести **по модулю 10⁹ + 7**.

### Входные данные
Во входных данных записаны два числа:
`N K`

Ограничения:
`0 ≤ K ≤ N ≤ 1000`

### Выходные данные
Выведите одно число — количество бинарных строк длины **N**, содержащих ровно **K единиц**, **по модулю 10⁹ + 7**.

### Ограничения
- Ограничение по времени: **1 секунда**
- Ограничение по памяти: **256 МБ**
