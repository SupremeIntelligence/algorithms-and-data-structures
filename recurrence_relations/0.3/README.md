# 0.3 Ones (Part 2)

## English version

### Problem
Given two integers **N** and **K**, determine how many **binary strings of length N** contain **exactly K ones**.

The result may be very large, so it must be printed **modulo 10⁹ + 7**.

### Input
Two integers:
`N K`

Constraints:
`0 ≤ K ≤ N`

### Output
Print a single integer — the number of binary strings of length **N** containing exactly **K ones**, **modulo 10⁹ + 7**.

### Constraints
- Time limit: **1 second**
- Memory limit: **256 MB**

# 0.3 Единицы (часть 2)

## Русская версия

### Условие
Даны два целых числа **N** и **K**.  
Необходимо определить, **сколько существует бинарных строк длины N**, содержащих **ровно K единиц**.

Так как ответ может быть очень большим, его необходимо вывести **по модулю 10⁹ + 7**.

### Входные данные
Два целых числа:
`N K`

Ограничения:
`0 ≤ K ≤ N`

### Выходные данные
Выведите одно число — количество бинарных строк длины **N**, содержащих ровно **K единиц**, **по модулю 10⁹ + 7**.

### Ограничения
- Ограничение по времени: **1 секунда**
- Ограничение по памяти: **256 МБ**
