# 0.0 Sum of Keys in a Binary Search Tree

## English version

### Problem
A sequence of integers is given — the keys of vertices inserted into a binary search tree in order.

All keys in a BST must be unique, so if a key that already exists in the tree appears in the input, it is ignored.

The task is to build the tree and compute the **sum of all keys** stored in it.

### Input
`input.txt` contains integers — one key per line — representing the order of insertion into the BST.

### Output
`output.txt` must contain a single integer — the **sum of all keys in the tree**.

### Constraints
- Time limit: **1 second**
- Memory limit: **not specified**
- Key range: **[-2³¹, 2³¹ − 1]**

# 0.0 Сумма ключей бинарного поискового дерева

## Русская версия

## Условие
Дана последовательность целых чисел — ключи вершин, добавляемые в бинарное поисковое дерево по порядку.

Все ключи в бинарном поисковом дереве уникальны, поэтому если во входных данных встречается ключ, который уже есть в дереве, он **игнорируется**.

Необходимо построить дерево и найти **сумму ключей всех его вершин**.

## Входные данные
Файл `input.txt` содержит целые числа — по одному ключу в строке — в порядке добавления в дерево.

## Выходные данные
Файл `output.txt` должен содержать одно число — **сумму всех ключей дерева**.

## Ограничения
- Ограничение по времени: **1 секунда**
- Ограничение по памяти: **не указано**
- Диапазон ключей: **[-2³¹, 2³¹ − 1]**
